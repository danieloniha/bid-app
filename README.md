PROJECT
HTML, CSS, JS & LARAVEL

REQUIREMENTS
- IDE
- Php 8.2 & Laravel 12.0 Installed

SETUP

- Create a database for the project and set up your env
- Run migrations and seeding
  php artisan migrate
  php artisan db:seed
  php artisan storage:link (for documents)

- Run the application
  php artisan serve
